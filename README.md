# LanBoard

🚀 **LanBoard** 是一个轻量级的局域网消息板工具，支持 **PC / 手机 / 平板** 在同一局域网内，通过浏览器实时发送文本和文件。

无需安装客户端，扫码即可使用。

---

## ✨ 功能特性

- 📡 局域网内实时消息同步（WebSocket）
- 📁 支持文件 / 图片上传与下载
- 📱 手机、平板、PC 浏览器通用
- 🔐 通过 URL 路径密码控制访问
- 🧹 支持一键清屏（同时清空文件）
- 🗑️ 单条消息删除（自动删除对应文件）
- 📂 上传文件按 **日期子目录** 自动整理
- 🖥️ Windows 托盘常驻，低打扰运行
- ⚡ 轻量、无需数据库、即开即用

---

## 🖼️ 使用场景

- 电脑 ↔ 手机 传文件
- 局域网临时共享文本 / 链接
- 内网设备之间的“剪贴板”
- 替代 AirDrop / QQ / 微信 的局域网传输

## 🚀 快速开始（Windows）

### 1️⃣ 下载

前往 **Releases** 页面下载最新版本：

👉 https://github.com/fhjd12/LanBoard/releases

解压后直接运行 `LanBoard.exe`

---

### 2️⃣ 启动

- 程序启动后会 **常驻系统托盘**
- 自动在程序目录生成：
  - uploads/ 文件夹：上传文件存放目录
  - data/ 文件夹：数据文件存放目录
  - config.json 配置文件

---

### 3️ 访问页面
在同一局域网的设备上，打开浏览器访问：

👉 http://<你的IP>:8787/<密码>    # 例如：http://192.168.1.100:8787/1234   (密码和端口可在 config.json 文件中修改)

也可以**扫码访问**（页面右上角二维码）

---

## ⚙️ 配置说明

首次运行会生成 `config.json`：

```json
{
  "app_name": "LanBoard",     // 程序名称
  "version": "1.1.0",        // 程序版本
  "host": "0.0.0.0",         // 监听地址
  "port": 8787,              // 监听端口
  "password": "1234",        // 访问密码
  "retention_hours": 24,     // 保留消息时间（小时）
  "clean_interval_hours": 24    // 清理间隔（小时）
  "max_file_mb": 30,       // 最大文件大小（MB）
  "history_limit": 800       // 历史消息条数限制
}
修改后重启生效。

## 🧠 技术实现

- 后端：FastAPI
- 通信：HTTP + WebSocket
- 前端：原生 HTML / JS（无框架）
- 打包：PyInstaller（onedir）
- 系统托盘：pystray

## 🔒 安全说明

- 仅适用于 可信局域网环境
- 密码通过 URL 控制访问
- 不建议暴露到公网使用
